Application Overview

This Application gives a graphical overview of any Weather application. Start with plotting a Pin on the Map. Click on the pin again, it will display the place name and temperature. There is an info button. On clicking that button, a new screen will open displaying the 5-day forecast. It will show the following details:

  • Temperature,

  • Humidity,

  • Rain in last 3hours,

  • Date

The following actions can be done in the main screen:

Bookmark: Places need to be bookmarked, so that the application will load those places by default when next time it will be launched.

Delete: Remove any bookmarked place.

Setting: You can reset all the Bookmarked places and even you can change the Temperature unit to Celcius or Fahrenheit as per your wish.

Help: Displays the application overview.
